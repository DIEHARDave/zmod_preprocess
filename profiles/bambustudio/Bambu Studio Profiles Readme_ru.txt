ПРИМЕЧАНИЕ: Эти профили не были протестированы в той же степени, что и профили 
для OrcaSlicer. Используйте на свой страх и риск!

Чтобы добавить эти профили в Bambu Studio:
- Сначала необходимо добавить профиль Creality K1 0.4mm в Bambu Studio.
- Затем их можно импортировать через Файл -> Импорт -> Импорт конфигураций.
- Стандартные профили процессов печати подходят только для сопла 0.4 мм; вам 
  потребуется создать собственные профили процессов для других размеров сопел.

Эти профили созданы для максимально точного воспроизведения стандартных профилей 
OrcaSlicer (не Orca-Flashforge) для AD5X, за исключением нескольких намеренных 
отличий:
- Добавлен G-код zMod для начала, завершения и смены слоя.
- Стартовый G-код изменен для автоматической установки USE_TRASH_ON_PRINT на 2.
- Добавлен G-код для смены филамента.
- Время загрузки филамента — 66 сек., время выгрузки — 0 сек.
- Тип Z-hop установлен на «нормальный» (normal).

Режим USE_TRASH_ON_PRINT = 0 (nopoop mode) не может быть использован в 
Bambu Studio. Данный профиль поддерживает только USE_TRASH_ON_PRINT = 2.

Не забудьте вернуть настройку USE_TRASH_ON_PRINT к нужному значению после 
использования этих профилей. Это можно игнорировать, если вы планируете 
использовать данные профили для всех своих задач печати.

** Примечание касательно «Очистки в заполнение», «Очистки в объект» и т.д.: **
Эти опции можно использовать с данными профилями, и они будут работать корректно. 
В Bambu Studio НЕТ той же ошибки, что и в OrcaSlicer, которая препятствует их 
использованию вне режима nopoop.

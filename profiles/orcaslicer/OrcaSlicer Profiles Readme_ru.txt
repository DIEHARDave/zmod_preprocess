Чтобы добавить эти профили в OrcaSlicer:
- Сначала необходимо добавить стандартные профили AD5X в OrcaSlicer (вы 
  должны добавить каждый размер сопла, который планируете использовать).
- Затем их можно импортировать через Файл -> Импорт -> Импорт конфигураций.
- Вы можете продолжать использовать свои существующие профили материалов и 
  процессов печати.

Отличия от стандартных профилей AD5X:
- Время разгона вентилятора установлено на 1.5 сек.
- Время пуска (kick-start) вентилятора установлено на 0.5 сек.
- Добавлен G-код zMod для начала, завершения и смены слоя.
- Стартовый G-код изменен для автоматической установки режима USE_TRASH_ON_PRINT.
- Добавлен G-код для смены филамента.
- Включена очистка в черновой башне (prime tower).
- Время загрузки филамента — 66 сек., время выгрузки — 0 сек.
- Тип Z-hop установлен на «нормальный» (normal).

G-коды начала печати и смены филамента настроены для поддержки режимов USE_TRASH_ON_PRINT=0 («режим без сброса»),
USE_TRASH_ON_PRINT=2 («режим сброса под управлением слайсера») и USE_TRASH_ON_PRINT=1 («стандартный режим сброса»).

Чтобы переключиться в режим сброса под управлением слайсера, отключите параметр «Очистка в башне прайминга» (Purge in prime tower)
в настройках мультиматериальной печати принтера.
Стандартный «режим сброса» принтера автоматически определяется и используется в том случае, если для типа пластины задано значение
«По объектам» (By object) и при этом включена опция «Очистка в башне прайминга», — поскольку в данном случае башня для очистки отсутствовать.

Не забудьте вернуть настройку USE_TRASH_ON_PRINT к нужному значению после 
использования этих профилей. Это можно игнорировать, если вы планируете 
использовать данные профили для всех своих
